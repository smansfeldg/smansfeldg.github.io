Investigación Aeroespacial

UNLaM Ground Segment (UGS)

Sistemas de comunicación satelital en tiempo real

Sobre el Proyecto

Proyecto de investigación desarrollado en el GIDSA (Grupo de Investigación y Desarrollo en Sistemas Aeroespaciales) de la Universidad Nacional de La Matanza, enfocado en la integración en tiempo real de sistemas de comunicación satelital mediante tecnologías web modernas.

Período: Marzo 2022 - Marzo 2024
Rol: Desarrollador de Software Aeroespacial
Modalidad: Colaborador - Híbrido

Logros Principales

  • Integración exitosa en tiempo real mediante WebSockets
  • Pruebas de estrés completadas en tecnología backend
  • Integración con API de red Satnogs
  • Actualización de integración con NASA OPENMCT

Tecnologías y Metodología

Stack Tecnológico
WebSockets NodeJS NASA OPENMCT Satnogs API Real-time Systems
Metodología de Trabajo
  • • Desarrollo ágil con sprints de 2 semanas
  • • Testing de carga y performance
  • • Documentación técnica detallada
  • • Colaboración con equipos internacionales

Impacto del Proyecto

El proyecto UGS representa un avance significativo en la capacidad de Argentina para el monitoreo y control de satélites en tiempo real. Mi contribución se centró en el desarrollo del backend, implementación de comunicación WebSocket bidireccional, y la optimización del rendimiento del sistema para manejar múltiples conexiones simultáneas.
